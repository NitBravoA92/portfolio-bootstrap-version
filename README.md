# 📖 Portfolio Website: Bootstrap version <a name="about-project"></a>

**Portfolio Website: Bootstrap version** is a prototype version of the Portfolio website created from scratch using the Bootstrap framework. This project is focused on testing the skills for web layout using the a CSS framework.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder-name
  git clone git@github.com:NitBravoA92/portfolio-bootstrap-version.git
```

### Prerequisites
In order to install, modify and run this project, it is necessary to have the following applications installed:
- **Git:** to manage the project versions of source code. [You can Download Git here](https://git-scm.com/)
- **Nodejs and NPM:** to install and manage the project dependencies. [Nodejs and NPM installation guide](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
- **A code editor** like Visual Studio Code (Recommended) or any other of your preference. It is necessary to open the project and add or modify the source code. [You can Download Visual Studio Code here](https://code.visualstudio.com/)

It is also important to have at least basic knowledge of HTML and CSS languages, and the fundamentals of Bootstrap framework, so you can understand and work more easily with the html code and the Bootstrap classes. 
- [Learn the basics of HTML](https://developer.mozilla.org/en-US/docs/Web/HTML)
- [Semantic HTML: What It Is and How to Use It Correctly](https://www.semrush.com/blog/semantic-html5-guide/)
- [Learn the basics of CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [The HTML5 input types](https://developer.mozilla.org/en-US/docs/Learn/Forms/HTML5_input_types)
- [Client-side form validation](https://developer.mozilla.org/en-US/docs/Learn/Forms/Form_validation)
- [Bootstrap 5 Tutorial](https://www.w3schools.com/bootstrap5/)
- [Bootstrap CSS Framework - Full Course for Beginners](https://www.youtube.com/watch?v=-qfEOE4vtxE)

If you want to make advanced customizations to the Bootstrap framework components used in this project, you need to learn about the SASS preprocessor and how to use it with Bootstrap.
- [Sass: Syntactically Awesome Style Sheets](https://sass-lang.com/)
- [Sass Tutorial](https://www.w3schools.com/sass/)

### Install

Install this project by running the next command into your project folder:

```sh
  npm install
```

### Usage

Open the HTML or CSS files and modify the internal code and then run the following command:

```sh
  npx run .
  npx stylelint "**/*.{css,scss}"
```
This will show you a log with details about errors (if any) and changes that would be necessary to solve those errors and improve the code.

**Note**: Please only modify the HTML and CSS files. Do not modify the configuration files of the project.
